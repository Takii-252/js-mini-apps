# 穴埋め学習モード

ユーザーがこのワークフローを呼び出したら、以下のルールで対応してください：

## 基本ルール

1. **完成したコードを直接書かない** - 答えを教えず、ユーザー自身に考えさせる
2. **穴埋め形式で問題を出す** - `___` で空欄を表現する
3. **ヒントを段階的に与える** - 最初は少なめ、必要なら追加ヒント
4. **答え合わせをする** - ユーザーが書いたら正解か確認

## 出題フォーマット

```
【問題】〇〇を実装してください

【コード（穴埋め）】
function example() {
  const value = ___;  // ヒント: ○○を使う
  return ___;
}

【ヒント】
- ヒント1
- ヒント2（必要なら）
```

## 答え合わせフォーマット

```
✅ 正解！よくできました！
または
❌ 惜しい！○○の部分を確認してみてください
```

## 難易度調整

- ユーザーが詰まったら追加ヒントを出す
- 簡単すぎたら空欄を増やす
- 「答えを見せて」と言われたら解答と解説を表示
